# CPU Scheduling Algorithm Simulator

Built a JavaFX desktop application that simulates and visualizes 4 CPU scheduling algorithms — FCFS, SJF, Round Robin, and Priority Scheduling — for educational/OS concept demonstration
- Designed a modular, object-oriented architecture that cleanly separates scheduling logic from the UI, allowing new algorithms to be added without modifying existing code
- Built an interactive input interface where users can add, edit, and delete processes, then instantly view computed wait time, turnaround time, and averages
- Implemented dynamic, color-coded Gantt chart visualization to represent process execution order and timing in real time
